# Week 2 Lab – Reflection

1. What is one thing JavaScript can do that HTML/CSS cannot?

JavaScript can create dynamic interactivity and logic, such as validating forms, calculating averages, handling button clicks, and modifying the DOM in response to user actions.

2. Why do we prefer const by default?

We prefer const because it prevents accidental reassignment. It makes code safer and easier to understand. We only use let when reassignment is necessary.

3. What is the difference between == and ===?

== compares values with type conversion.
=== compares both value and type without conversion.
=== is safer and more predictable.

4. What was the most useful Console command today (and why)?

console.table() was very useful because it clearly displays arrays of objects in a readable table format.

5. CSS hover vs JS hover: which felt more robust and why?

CSS hover felt more robust and simpler because it requires less code and is declarative. JavaScript hover requires event listeners and more logic.
